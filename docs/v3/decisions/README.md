# v3 Decision Records

These records convert v3 open questions into reviewable protocol decisions
before schema and validator work claims compatibility.

Each record identifies owner, scope, options, recommendation, and accepted
outcome or blocking state. Open discussions remain useful background, but these
records are the execution authority for v3 implementation.
